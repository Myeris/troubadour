Personal information will be requested only in cases where such information is necessary to identify you or contact you. Troubadour may disclose personal information only if required by law or where there is a firm belief that such action is necessary to:

Access to personal information

The personal information you provide to Troubadour for recording, or for other purposes are not shared with third parties without the express permission of the user, except in limited cases described above. Personal information will always be used for the purposes described above. Troubadour provides users with all the tools you need to make sure that personal information is accurate and up to date. If you register for services offered by Troubadour, you can edit, update or delete your data at any time.
